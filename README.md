# HR Analytics Dashboard using Python & Power BI

## 📌 Overview
This project was completed as part of the CodeAlpha Data Analytics Internship.  
The project focuses on analyzing HR employee data using Python and Power BI to identify employee attrition trends and workforce insights.

---

# ✅ Tasks Completed

## Task 2 – Exploratory Data Analysis (EDA)

Performed data analysis using Python to:
- Check missing and duplicate values
- Analyze employee attrition
- Study department-wise employee distribution
- Analyze salary and age patterns
- Generate statistical insights and graphs

### Tools Used
- Python
- Pandas
- Matplotlib

---

## Task 3 – Data Visualization

Created an interactive HR Analytics Dashboard using Power BI.

### Dashboard Includes
- Employee Count
- Attrition Count & Rate
- Salary Analysis
- Age Group Analysis
- Department Analysis
- Gender Analysis
- Job Role Analysis

---

# 📊 Key Insights

- Employees aged 26–35 show higher attrition.
- Employees with lower salaries leave more frequently.
- Some job roles have significantly higher attrition rates.
- Attrition decreases as years at company increase.
